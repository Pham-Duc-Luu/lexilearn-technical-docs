## 必要条件

Node.js バージョン 18.0 以上（`node -v` で確認可能）。  
複数の Node バージョンを管理したい場合は、`nvm` の利用をおすすめします。

## 🚀 フロントエンドプロジェクトのセットアップ

Git をインストールした後、以下のコマンドでフロントエンドのソースコードをクローンします：

```bash
git clone https://github.com/Pham-Duc-Luu/Lexilearn-front-end-v2.git
```

クローン後、依存パッケージをインストールします：

```bash
cd ./Lexilearn-front-end-v2
```

```bash
npm install --force
```

## プロジェクトの中核

### 🎨 UI ライブラリ

このプロジェクトでは以下の UI ライブラリを使用しています：

- **[NextUI](https://nextui.org/):** 美しくモダンな React コンポーネントライブラリ。ダークモード対応でカスタマイズも簡単。
- **[shadcn/ui](https://ui.shadcn.com/):** Tailwind CSS と Radix UI を組み合わせたオープンソースの柔軟な UI コンポーネント集。
- **[Framer Motion](https://www.framer.com/motion/):** スムーズで洗練されたアニメーションを実現できる強力な React アニメーションライブラリ。

### 🧠 状態管理（State Manager）

プロジェクトでは状態管理ライブラリとして **[Redux Toolkit](https://redux-toolkit.js.org/)** を使用しています。

Redux Toolkit を利用することで、`store`、`reducer`、`action`、`非同期処理（createAsyncThunk）` の作成が簡潔になり、保守性が高く分かりやすい構造になります。

> 💡 デバッグや開発を効率化するために、**Redux DevTools Extension** のインストールをおすすめします：
>
> 🔗 [Redux DevTools for Chrome](https://chrome.google.com/webstore/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd)
